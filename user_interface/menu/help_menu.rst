.. _help-find_action:

Find Action
-----------

.. .. versionadded:: 22.04

This feature allows you to search for any command or action available. Simply enter the search term and then select the command or action you want to execute. Kdenlive uses fuzzy search or look-ahead search, so that every character you type changes the search result on the fly.

.. figure:: /images/user_interface/menu_reference/kdenlive2304_find_action.webp
   :align: left
   :width: 650px

.. _help-copy_Debug-Information:

Copy Debug Information
----------------------

This function copies relevant debug information into the clipboard. When you report a bug in Bugzilla you can paste this information in the section `SOFTWARE/OS VERSIONS`


.. _help-report_bug:

Report Bug
----------

This function allows you to report a bug directly into the Kdenlive Bugzilla bugtracker by clicking on :menuselection:`Launch Bug Report Wizard`.


.. _help-donate:

Donate
------

This function allows you to donate. It points you to the Kdenlive |donate| page where the donation get collected for Kdenlive (https://kdenlive.org/fund/).


.. _help-about_kdenlive:

:guilabel:`About Kdenlive` This function shows you all details about Kdenlive.


.. _help-about_kde:

:guilabel:`About KDE` Kdenlive is part of KDE. It shows you more details about KDE.
