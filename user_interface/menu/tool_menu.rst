Tool Menu
=========

.. contents::

.. image:: /images/Kdenlive_tool_menu_selection_kde.png
  :align: center
  :alt: Tool menu selection


The options on this menu provide three modes and six tools which affect how operations are performed on clips in the timeline. These same options can also be accessed from :ref:`timeline_edit_modes` and :ref:`timeline_edit_tools`. More details on their usage can be found there.
