Editing
-------



.. list-table::
  :header-rows: 1

  * -  Action 
    -  Shortcut
  * -  Make a cut (make sure track is selected first) 
    -  :kbd:`Shift` + :kbd:`R` 
  * -  Play 
    -  :kbd:`Space` 
  * -  Play zone 
    -  :kbd:`Ctrl` + :kbd:`Space` 
  * -  Render 
    -  :kbd:`Ctrl` + :kbd:`Return` 
  * -  Switch Monitor 
    -  :kbd:`T` 
  * -  Forward 
    -  :kbd:`L` 
  * -  Rewind 
    -  :kbd:`J` 
  * -  Forward 1 frame 
    -  :kbd:`Right` 
  * -  Rewind 1 frame 
    -  :kbd:`Left` 
  * -  Forward 1 second 
    -  :kbd:`Shift` + :kbd:`Right` 
  * -  Rewind 1 second 
    -  :kbd:`Shift` + :kbd:`Left` 
  * -  Toggle Full Screen Mode on and off 
    -  :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`F` 
  * -  Go to Clip End 
    -  :kbd:`End` 
  * -  Go to Clip Start 
    -  :kbd:`Home` 
  * -  Go to Next Snap Point 
    -  :kbd:`Alt` + :kbd:`Right` 
  * -  Go to Previous Snap Point 
    -  :kbd:`Alt` + :kbd:`Left` 
  * -  Go to Project End 
    -  :kbd:`Ctrl` + :kbd:`End` 
  * -  Go to Project Start 
    -  :kbd:`Ctrl` + :kbd:`Home` 
  * -  Go to Zone End 
    -  :kbd:`Shift` + :kbd:`O` 
  * -  Go to Zone Start 
    -  :kbd:`Shift` + :kbd:`I` 
  * -  Group Clips 
    -  :kbd:`Ctrl` + :kbd:`G` 
  * -  Set Zone In 
    -  :kbd:`I` 
  * -  Ungroup Clips 
    -  :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`G` 
  * -  Set Zone Out 
    -  :kbd:`O` 
  * -  :ref:`editing` 
    - :kbd:`Ctrl` + :kbd:`X` [1]_   


.. [1]  available in bleeding edge version > 0.9.10 (Jan2015)
