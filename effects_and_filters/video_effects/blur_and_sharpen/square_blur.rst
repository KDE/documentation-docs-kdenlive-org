.. rubric:: Description

This effect blurs the entire clip. **Kernel size** determines the blurriness.


.. rubric:: Parameters

.. list-table::
   :header-rows: 1
   :width: 100%
   :widths: 20 10 70
   :class: table-wrap

   * - Parameter
     - Value
     - Description
   * - Kernel size
     - Integer
     - The size of the kernel, as a proportion to its coverage of the image. Range is 0 - 500, default is 50.
