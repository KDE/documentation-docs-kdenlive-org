.. rubric:: Description

This effect/filter negates (inverts) the input video or its alpha channel.

.. list-table::

   :width: 100%
   :widths: 20 10 70
   :class: table-wrap

   * - Parameter
     - Value
     - Description
   * - Negate alpha
     - Switch
     - If checked, it negates/inverts the alpha component (if present). Default is **off**.
