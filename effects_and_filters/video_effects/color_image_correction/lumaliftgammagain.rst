.. rubric:: Description

This effect/filter can be used to apply lift, gain and :term:`gamma` correction to :term:`luma` values of the source image.


.. rubric:: Parameters

.. list-table::
   :header-rows: 1
   :width: 100%
   :widths: 20 10 70
   :class: table-wrap

   * - Parameter
     - Value
     - Description
   * - Lift
     - Integer
     - Adjusts the darker areas
   * - Gain
     - Integer
     - Adjusts the brighter areas
   * - Gamma
     - Integer
     - Adjusts the midtone areas
